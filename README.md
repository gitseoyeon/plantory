# 🌱 Plantory – 식물 관리 및 정보 공유 플랫폼

> **Plant + Story**  
> 나의 반려식물의 성장기를 기록하고, 식물 정보를 탐색하고, 커뮤니티에서 소통할 수 있는 식물 관리 플랫폼입니다.  
> 📅 개발 기간: 25.10.13 - 25.10.27

---

## 🌿 프로젝트 소개

**Plantory(플랜토리)** 는 다음을 목표로 개발된 식물 관리 플랫폼입니다

- 반려 식물을 **체계적으로 관리**하고 성장 과정을 기록
- AI 기반 식물 식별과 식물 백과 검색을 통해 **정확한 정보 탐색** 
- 사용자 간 식물 관련 소통을 지원하는 **커뮤니티 제공**  
- 실시간 알림과 QR 코드를 이용한 **편의성 있는 UX 제공**

이 플랫폼은 *식물 관리 + 식물 정보 탐색 + 커뮤니티 기능*을 결합한 **풀 패키지 식물 서비스**입니다.
